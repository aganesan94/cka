# YAML Tutorial

## Key Value Pairs

```yaml
Fruit: Apple
Vegetable: Carrot
Liquid: Water

```

## Array/List

* Ordered

```yaml

- Apple
- Mango
- Papaya

- Carrot
- Beetroot
```

